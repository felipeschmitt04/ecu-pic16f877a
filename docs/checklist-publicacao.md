# Checklist de Publicacao

Antes de divulgar no GitHub/LinkedIn:

- [ ] Adicionar imagens ou prints em `docs/imagens/`.
- [ ] Adicionar um video curto ou GIF da simulacao funcionando.
- [ ] Conferir se o README abre corretamente no GitHub.
- [ ] Verificar se arquivos gerados nao entraram no commit.
- [ ] Confirmar que o texto deixa claro que e um projeto didatico.
- [ ] Rodar ERC/DRC no KiCad, se o KiCad estiver disponivel.
- [ ] Compilar novamente no MPLAB X/XC8, se o ambiente estiver disponivel.
