# Sugestao de Post para LinkedIn

Durante a disciplina de Microprocessadores e Microcontroladores, desenvolvi uma
ECU didatica baseada no PIC16F877A, integrando firmware em C, simulacao de
circuito e projeto de PCB.

O projeto simula o controle basico de um motor de combustao, com leitura ADC de
sensores como TPS, RPM, sonda lambda e temperatura, alem de mapas de injecao,
controle de bicos, ignicao, bomba de combustivel, radiador, motor de partida e
interface LCD 16x2.

Tambem desenvolvi a simulacao no Proteus e o esquematico/PCB no KiCad, conectando
a parte de software embarcado com a parte de hardware.

Foi um projeto muito importante para consolidar conceitos de timers,
interrupcoes, ADC, EEPROM, atuadores e arquitetura de sistemas embarcados. O
trabalho recebeu nota maxima na disciplina e agora esta documentado como parte
do meu portfolio academico.

Tecnologias usadas:

- C
- PIC16F877A
- MPLAB X / XC8
- Proteus
- KiCad
- LCD 16x2
